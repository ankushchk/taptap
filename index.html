taptap 🦀

An ultra-lightweight, high-performance mechanical keyboard sound simulator CLI written in Rust.

View on GitHub

Features

  • Sub-Millisecond Latency: Pre-decodes audio files into raw PCM buffers at startup, completely removing runtime disk I/O and decoding overhead.
  • Concurrently Mixed Overlaps: Audio slices are routed directly to the system hardware mixer. Typing fast produces natural, overlapping switch sounds (polyphony).
  • Ultra Lightweight: Zero heavy dependencies, compiling down to under 800 KB and utilizing less than 10MB of RAM.
  • Soundpack Discovery: Automatically scans folders in your local soundpacks/ directory and falls back to checking ~/.taptap/soundpacks/.

Installation

To install taptap globally on macOS, Linux, or Windows (Git Bash), run the following command in your terminal:

curl -fsSL https://raw.githubusercontent.com/ankushchk/taptap/master/install.sh | sh

Quick Start & Usage

Run the program from any directory:

taptap

CLI Commands & Flags

  • taptap help: Show the help menu
  • taptap list: List all installed soundpacks
  • taptap play <index> [optional_volume_0_100]: Play a soundpack by index directly (e.g., taptap play 1 90)

Adding Custom Soundpacks

You can add custom soundpacks by dropping their folders into either the local soundpacks/ directory or the global ~/.taptap/soundpacks/ directory. Each folder must contain:

  1. config.json: The mapping of scan codes to sound clips (Mechvibes standard).
  2. sound.ogg: The combined audio sprite file containing all sound clips.

Uninstallation

To completely remove taptap and all its downloaded soundpacks from your system, run the following commands in your terminal:

sudo rm /usr/local/bin/taptap
rm -rf ~/.taptap
